Площадка для публикации своих данных и для ознакомления с информацией о других пользователях



Функции:
1)Возможность создания аккаунта
2)Возможность восстановления аккаунта
3)Возможность входа в аккаунт
4)Сохранение последних входов при нажатии соответствующей галочки
5)Возможность редактирования страницы аккаунта(аватарка и информация о пользователе)
6)Возможность смотреть страницы других пользователей
7)Возможность оставлять комментарии другим пользователям
8)Возможность смены шрифта на главной странице(для каждого пользователя свой)
9)Возможность смены аккаунта


Кратко про классы и их методы:
class Qwidget: (изменение начального шрифта)
	def center: (функция перемещения окна в центр экрана)
	def closeEvent: (при закрытии файла закрывается соединение с дб)


class Authentication: (окно авторизации в систему)
	def tryLogon: (попытка входа в систему)
	def forgotPassword: (открытие окна, отвечающего за замену пороля при его забытии)
	def registration: (открытие окна, отвечающего за регистрацию)


class Registration: (само окно регистрации)
	def tryRegistration: (попытка зарегистрироваться)


class ForgotPassword: (само окно восстановления пароля)
	def Recover1: (шаг восстановления - 1)
	def Recover2: (шаг восстановления - 2)
	def Recover3: (шаг восстановления - 3)
	def Recover4: (шаг восстановления - 4)


class MainWidget: (главное окно, открытие MainTabWidget)


class MainTabWidget: (главный виджет главного окна - QTabWidget)
	def tab1UI: (1 вкладка вкладка с профилем пользователя)
	def tab2UI: (2 вкладка с нахождением информации других пользователей и оставлением комментариев другим пользователям)
	def tab3UI: (3 вкладка с редактирование профиля)
	def update1: (обновление информации на 1 вкладке)
	def update2: (обновление информации на 2 вкладке)
	def changeProfile: (нахождение информации других пользователей во 2 вкладке)
	def giveComment: (оставить комментарий на странице выбранного профиля)
	def changeAva: (смена аватарки)
	def saveChange: (сохранение изменений)
